Welcome to my personal web project. The theme is inspired by the Red Dead Redemption 2 (RDR2) Pause Menu, but the functionality is built for survival in the world of academia. This project is currently Under Construction (basically as messy as the Van der Linde camp after a Pinkerton raid). I'm planning to polish it soon... once I get some faith!  

🐴 Features (The Camp Upgrades)Despite the "random" aesthetic, this hub is packed with functional tools powered by custom logic:
⌚ Interactive Pocket Watch: A RDR2-style analog watch that rotates in real-time based on your local clock.  
🗺️ Road Map & Academic Checklist: A zoomable map interface featuring a dynamic To-Do List to track your courses or "missions".  
📈 Progress Tracker (SKS & GPA): Arthur Morgan might not care about grades, but you do. This tool features an automatic SKS and GPA weighted calculator. 
📅 Bounty Board Calendar: A comprehensive calendar to log assignments and events. It includes custom themes (Classic, Blood, Sand), reminder notifications, and data export/import functionality.  
🔗 Help & Custom Links: A card-based bookmark system for your important links, from the RDR2 Wiki to university portals.  

🛠️ The Armory (Tech Stack)This project is built using the "Holy Trinity"—no heavy frameworks, just pure horsepower:  
HTML5: The skeletal structure, as sturdy as a Shire horse.  
CSS3: Custom styling to achieve that gritty, Western aesthetic.  Vanilla JavaScript: The brains behind the operation, handling all the interactive logic.  
LocalStorage API: Ensures your calendar, GPA data, and checklists stay saved on your device without needing a backend.  

⚠️ Bounty NoticeThe UI is currently optimized for desktop; if you open this on a phone and it looks like a stagecoach wreck, that's just the "retro" charm.  If you find any bugs or buttons that don't work, consider it a "lore-accurate" feature of the late 1800s.
